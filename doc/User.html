    class User

    User model representing system users with authentication and API access

    Users belong to companies in a multi-tenant architecture. Each user has secure password authentication and an automatically generated API key for programmatic access to the system.

    @example Creating a new user

    company = Company.find(1)
user = User.create!(
  email_address: 'john@example.com',
  password: 'secure_password',
  company: company
)
